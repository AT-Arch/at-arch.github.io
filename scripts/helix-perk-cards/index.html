Overview

A modular system for Helix that allows you to add, manage, and customize perk cards for your server. Designed for flexibility and easy integration with your existing Helix setup.

Planned Features

  • Lightweight perk registration API
  • UI component hooks
  • Configurable rarity and progression
  • Extensible effects system
  • Localization-ready strings

Status

This project is currently in active development. Target planned version: 0.1.0.
